# フライングシャーライン 3Dシミュレータ

アルミニウム製造ラインの板切断工程「フライングシャー（走間剪断）」の動作を再現する、
ブラウザだけで動く 3D HTML アプリケーションです。

## 使い方

`index.html` をブラウザで開くだけで動作します（外部ライブラリ・ネットワーク接続は不要）。

```
open index.html        # macOS
start index.html       # Windows
```

## 再現しているライン構成（材料の流れ順）

1. **アンコイラー** — コイル状のアルミ材を巻き出す（コイル径は消費に応じて減少・自動交換）
2. **レベラー（矯正機）** — 千鳥配置のロールで巻きぐせ・反りを矯正
3. **測長ピンチロール** — 板に密着して送り長さを計測
4. **フライングシャー** — 板と同じ速度で走行しながら切断する走間剪断機
5. **ランアウトコンベア** — ライン速度より高速で搬送し、切板間にギャップを形成
6. **スタッカー** — ストッパで定位置停止させて積載（8枚でパレット搬出）

## 動作サイクル

① 材料送り（測長）→ ② キャリッジ同期加速 → ③ 走間切断（速度同期率100%）→ ④ 復帰

切断タイミングはキャリッジの加速距離・整定時間・刃の下降時間から逆算しており、
設定した切断長さ通りに切断されます（HUDの「直前の切断長」で確認可能）。

## 主な機能

- ライン速度（30〜150 m/min）・切断長さ（2.0〜4.5 m）の変更
- アニメーション速度切替（0.25× / 0.5× / 1×）・一時停止
- カメラ操作（ドラッグ回転・ホイール/ピンチズーム）と視点プリセット（全体／シャー部／搬出・積載／上面）
- 機器ラベル表示、工程フェーズ・測長・切断枚数などのリアルタイムHUD
- 依存ライブラリなし：Canvas 2D 上の自作簡易3Dレンダラ（ペインタ法）で描画

※板厚・寸法・速度は動作を見やすくするため誇張しています。
