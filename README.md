# フライングシャーライン 3Dシミュレータ

アルミニウム製造ラインの板切断工程「フライングシャー（走間剪断）」の動作を再現する、
ブラウザだけで動く 3D HTML アプリケーションです。

## 使い方

`index.html` をブラウザで開くだけで動作します。
Three.js（MIT License）をファイル内に同梱しているため、外部ライブラリの取得や
ネットワーク接続は不要です。

```
open index.html        # macOS
start index.html       # Windows
```

## 再現しているライン構成（材料の流れ順）

1. **アンコイラー** — コイル材を外層から接線方向に巻き出す。コイル径は消費に応じて縮小し、
   緩み押さえの**スナバロール**が外周に幾何学的に追従（接触角を毎フレーム解析）
2. **レベラー（矯正機）** — 千鳥配置のロールで巻きぐせ・反りを矯正（手前側はカットアウェイ表示）
3. **測長ピンチロール** — 板に密着して送り長さを計測
4. **フライングシャー** — 板と同速で走行するキャリッジ上で、**クランプが板を固定**し、
   上刃・下刃の**すれ違い（シャー角配置）**で剪断。油圧シリンダのロッドも刃と連動
5. **ランアウトコンベア** — ライン速度より高速で搬送し、切板間にギャップを形成
6. **スタッカー** — ストッパ手前で減速・停止させ積載。8枚満載でパイルを
   払い出しチェーンコンベアへ横搬出

## 動作サイクル

① 材料送り（測長）→ ② キャリッジ同期加速 → ③ クランプ・走間切断 → ④ 復帰

切断タイミングは、同期加速距離・整定時間・刃先が板上面に達するまでの時間から逆算しており、
設定した切断長さ通りに切断されます（HUDの「直前の切断長」で確認可能）。

## 物理整合のポイント

- WebGL の深度バッファによる描画で、部材同士の貫通表現なし（全ロールは板に接触のみ）
- 板はコイル下端から**水平接線**で巻き出され、コイル径変化に応じてベジエ曲線でたわむ
- 全ロール・コイル・台車車輪は接触面の線速度が一致する向き・角速度で回転
- 切断は刃先が板上面へ達した瞬間に発生し、下降中の刃は材料と干渉しない
- 切板は追突判定つきで搬送され、ストッパ手前で減速して停止・落下積載
- 搬出中のパイルはサイドガイドの下をくぐり、チェーンコンベア上を移動

## 主な機能

- ライン速度（30〜150 m/min）・切断長さ（2.0〜4.5 m）の変更
- アニメーション速度切替（0.25× / 0.5× / 1×）・一時停止
- カメラ操作（ドラッグ回転・ホイール/ピンチズーム）、視点プリセット
  （全体／シャー部／搬出・積載／上面）、**キャリッジ追従視点**
- 機器ラベル表示、影の表示切替、工程フェーズ・測長・切断枚数などのリアルタイムHUD

※板厚・寸法・速度は動作を見やすくするため誇張しています（1目盛=1m相当）。
